<Heading level="2">
    Basic Syntax
</Heading>

<Paragraph>
    All miniMAL programs are valid JSON arrays representing S-expressions:
</Paragraph>

<Paragraph blockColor="light_grey">
<Paragraph>
    <Mark>["function_name", arg1, arg2, ...]</Mark>
</Paragraph>
</Paragraph>

<Heading level="2">
    String Functions
</Heading>

<Paragraph blockColor="light_grey">
<Paragraph>
    <Mark>["pr-str", ["`", "hello"], ["`", "world"]]</Mark>
</Paragraph>
<Paragraph>
    <Mark>=> "\"hello\" \"world\""</Mark>
</Paragraph>
<Paragraph>
    <Mark>["str", ["`", "hello"], ["`", "world"]]</Mark>
</Paragraph>
<Paragraph>
    <Mark>=> "helloworld"</Mark>
</Paragraph>
<Paragraph>
    <Mark>["prn", ["`", "hello"], 42]</Mark>
</Paragraph>
<Paragraph>
    <Mark>Output: "hello" 42</Mark>
</Paragraph>
</Paragraph>

<Heading level="2">
    Working with Lists
</Heading>

<Paragraph blockColor="light_grey">
<Paragraph>
    <Mark>["def", "nums", ["list", 1, 2, 3]]</Mark>
</Paragraph>
<Paragraph>
    <Mark>["map", ["fn", ["x"], ["*", 2, "x"]], "nums"]</Mark>
</Paragraph>
<Paragraph>
    <Mark>=> [2, 4, 6]</Mark>
</Paragraph>
<Paragraph>
    <Mark>["nth", "nums", 1]</Mark>
</Paragraph>
<Paragraph>
    <Mark>=> 2</Mark>
</Paragraph>
<Paragraph>
    <Mark>["first", "nums"]</Mark>
</Paragraph>
<Paragraph>
    <Mark>=> 1</Mark>
</Paragraph>
<Paragraph>
    <Mark>["rest", "nums"]</Mark>
</Paragraph>
<Paragraph>
    <Mark>=> [2, 3]</Mark>
</Paragraph>
</Paragraph>

<Heading level="2">
    Working with Hash Maps
</Heading>

<Paragraph blockColor="light_grey">
<Paragraph>
    <Mark>["def", "hm", {"a": 1, "b": 2}]</Mark>
</Paragraph>
<Paragraph>
    <Mark>["get", "hm", ["`", "a"]]</Mark>
</Paragraph>
<Paragraph>
    <Mark>=> 1</Mark>
</Paragraph>
<Paragraph>
    <Mark>["contains?", "hm", ["`", "b"]]</Mark>
</Paragraph>
<Paragraph>
    <Mark>=> true</Mark>
</Paragraph>
<Paragraph>
    <Mark>["keys", "hm"]</Mark>
</Paragraph>
<Paragraph>
    <Mark>=> ["a", "b"]</Mark>
</Paragraph>
<Paragraph>
    <Mark>["vals", "hm"]</Mark>
</Paragraph>
<Paragraph>
    <Mark>=> [1, 2]</Mark>
</Paragraph>
</Paragraph>
