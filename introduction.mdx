<Heading level="1">
    What is miniMAL?
</Heading>

<Paragraph>
    <Mark bold>miniMAL</Mark> (Minimal Lisp) is a delightful diminutive Lisp interpreter implemented in less than <Mark bold>1024 bytes</Mark> of JavaScript. It is Clojure-inspired yet significantly different in syntax and functionality, using <Mark bold>JSON</Mark> as its source code format.
</Paragraph>

<Heading level="2">
    Key Features
</Heading>

<BulletedList>
    <BulletedList>
        <Mark bold>JSON Source</Mark>: Programs are written in plain JSON format
    </BulletedList>
    <BulletedList>
        <Mark bold>Tail-Call Optimization</Mark>: Efficient recursion without stack overflow
    </BulletedList>
    <BulletedList>
        <Mark bold>Macros</Mark>: Write new syntactic structures with macro support
    </BulletedList>
    <BulletedList>
        <Mark bold>JavaScript Interop</Mark>: Direct access to JavaScript objects and methods
    </BulletedList>
    <BulletedList>
        <Mark bold>Error Handling</Mark>: try/catch/throw style exception handling
    </BulletedList>
    <BulletedList>
        <Mark bold>Multi-language</Mark>: Implementations in JavaScript, Python, and ClojureScript
    </BulletedList>
</BulletedList>

<Heading level="2">
    Quick Example
</Heading>

<Paragraph>
    Here is a simple miniMAL program:

</Paragraph>
