<Heading level="2">
    Core EVAL Function
</Heading>

<Paragraph>
    The Python version uses a similar <Mark>EVAL</Mark> function with TCO using <Mark>while True</Mark> loop:
</Paragraph>

<Paragraph blockColor="light_grey">
<Paragraph>
    <Mark>def EVAL(ast, env):</Mark>
</Paragraph>
<Paragraph>
    <Mark>    while True:</Mark>
</Paragraph>
<Paragraph>
    <Mark>        if type(ast) != list:</Mark>
</Paragraph>
<Paragraph>
    <Mark>            return getattr(env, ast) if type(ast) == str else ast</Mark>
</Paragraph>
<Paragraph>
    <Mark>        # Apply forms...</Mark>
</Paragraph>
</Paragraph>

<Heading level="2">
    Environment Class
</Heading>

<Paragraph blockColor="light_grey">
<Paragraph>
    <Mark>def Env(outer=object, binds=[], exprs=[], d=None):</Mark>
</Paragraph>
<Paragraph>
    <Mark>    d = d or {}</Mark>
</Paragraph>
<Paragraph>
    <Mark>    for i in range(len(binds)):</Mark>
</Paragraph>
<Paragraph>
    <Mark>        if binds[i] == "&":</Mark>
</Paragraph>
<Paragraph>
    <Mark>            d[binds[i+1]] = exprs[i:]</Mark>
</Paragraph>
<Paragraph>
    <Mark>            break</Mark>
</Paragraph>
<Paragraph>
    <Mark>    return type("Env", (outer,), d)</Mark>
</Paragraph>
</Paragraph>

<Paragraph>
    Uses Python's <Mark>type()</Mark> to create dynamic classes with prototype chain for scope lookup.
</Paragraph>

<Heading level="2">
    Special Forms
</Heading>

<Heading level="3">
    def
</Heading>

<Paragraph blockColor="light_grey">
<Paragraph>
    <Mark>if "def" == ast[0]:</Mark>
</Paragraph>
<Paragraph>
    <Mark>    setattr(env, ast[1], EVAL(ast[2], env))</Mark>
</Paragraph>
<Paragraph>
    <Mark>    return getattr(env, ast[1])</Mark>
</Paragraph>
</Paragraph>

<Heading level="3">
    fn (with TCO)
</Heading>

<Paragraph blockColor="light_grey">
<Paragraph>
    <Mark>if "fn" == ast[0]:</Mark>
</Paragraph>
<Paragraph>
    <Mark>    fn = lambda *a: EVAL(ast[2], Env(env, ast[1], [*a]))</Mark>
</Paragraph>
<Paragraph>
    <Mark>    fn.A = [ast[2], env, ast[1]]  # For TCO</Mark>
</Paragraph>
<Paragraph>
    <Mark>    return fn</Mark>
</Paragraph>
</Paragraph>

<Heading level="3">
    if (TCO)
</Heading>

<Paragraph blockColor="light_grey">
<Paragraph>
    <Mark>if "if" == ast[0]:</Mark>
</Paragraph>
<Paragraph>
    <Mark>    ast = ast[2] if EVAL(ast[1], env) else (*ast[3:4], None)[0]</Mark>
</Paragraph>
<Paragraph>
<Paragraph>
    <Mark>    # TCO: loop continues with new ast</Mark>
</Paragraph>
</Paragraph>

<Heading level="2">
    Key Differences from JS Version
</Heading>

<BulletedList>
    <BulletedList>
        Uses <Mark>getattr/setattr</Mark> for environment lookups
    </BulletedList>
    <BulletedList>
        Dynamic class creation with <Mark>type()</Mark>
    </BulletedList>
    <BulletedList>
        <Mark>builtins.__dict__</Mark> for Python interop
    </BulletedList>
    <BulletedList>
        <Mark>sys.argv[2:]</Mark> for command-line arguments
    </BulletedList>
</BulletedList>
